# Getting Started

Follow the jekyll installation guide:

https://help.github.com/en/articles/setting-up-your-github-pages-site-locally-with-jekyll

Example:

```bash
# install
bundle install --path vendor/bundle

# server it locally
bundle exec jekyll serve --watch
```
